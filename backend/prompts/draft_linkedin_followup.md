# Draft: LinkedIn follow-up bump

A short, no-pressure nudge sent a few days after the first DM got no reply. They already
received the opener, so do NOT re-pitch or re-explain — just a light, human bump that makes
it easy to respond or to pass.

## Hard constraints

- **≤ 300 characters.** Shorter is better — this is a bump, not a pitch.
- Lowercase-casual, contractions, no em-dashes, plain text.
- **ONE soft re-ask.** No links. No new pitch. No attachments.
- Reference that you reached out before, lightly. Always give an easy out.

## Structure

```
<one friendly line acknowledging the earlier note + something specific to their world>
<the same soft ask, lower-key, with an explicit easy out>
```

## Examples (target voice)

✅
> hey {{first_name}}, no worries if now's not the time — still genuinely curious how you run
> {{their thing}} day to day. happy to keep it to 10 min if that's easier. either way, all good.

✅
> {{first_name}}, circling back once in case my note got buried. not selling anything, just
> trying to learn how shops like yours handle it. open to a quick chat? no stress if not.

## Rules

- Never guilt-trip ("just following up AGAIN", "did you see my message?"). Keep it warm + low-pressure.
- Do NOT repeat the original pitch — assume they read it.
- One message, one ask. First-name sign-off optional.
- Match the active campaign's Offer/voice in the system prompt; the examples show tone, not domain.

## Output format

Return ONLY the message text — no preamble, no quotes, no subject line.
