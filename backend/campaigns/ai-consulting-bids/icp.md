# Ideal opportunity profile — software / AI-agent work worth bidding

We're triaging *work*, not people. The ideal opportunity is a **software build with an AI/LLM
or automation core that a solo/small AI-consulting shop can win and ship**. Score each posting
against this.

## Segments (label which one)

- **ai_agent_build** — the deliverable is an AI agent, LLM app, RAG system, chatbot, or
  workflow automation. Highest fit. This is exactly what we do.
- **ml_data** — machine learning, data pipelines, NLP, model integration. High fit.
- **software_build** — general custom software / API / integration work with a plausible AI
  angle we can propose. Medium–high fit.
- **gov_it_services** — a federal/SLED solicitation for software/IT services under a
  small-business set-aside we're eligible for. Medium fit; weight eligibility heavily.
- **out_of_icp** — none of the above.

## Strong-fit signals (raise the score)

- Explicit AI / LLM / agent / automation / RAG / chatbot / ML / NLP language
- Clear, bounded scope a solo could deliver in weeks-to-a-few-months
- Budget or rate that supports real work (favor fixed >$5k or hourly >$60; gov sub-$250K buys)
- Remote / no on-site requirement
- For gov: a **small-business set-aside** (Total Small Business, 8(a), SDVOSB, WOSB, HUBZone),
  NAICS in 5415xx / 518210, and a response window long enough to prepare a real bid

## Disqualifiers (score → below 40)

- **Not software** — hardware, staffing/body-shop, advisory-only, design-only, content, data
  entry, or "IT" that's really helpdesk/managed-services with no build
- **Ineligible for a solo LLC** — requires a facility/personnel security clearance, a large
  past-performance record, incumbent-locked, or team/prime-of-primes scale
- On-site / relocation required
- Deadline passed or a response window too short to prepare
- Obvious low-budget churn ($5/hr races to the bottom, "simple 1-hour fix" bait)
- Crypto/MLM/"revolutionary idea, equity only" red flags
